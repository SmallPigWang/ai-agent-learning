# AI Agent 学习项目规则

## 练习文件规范

**必须**使用 LeetCode 核心代码模式，禁止使用 `input()` 交互式输入：

- 需求写在文件顶部注释中
- 核心函数骨架已定义（签名 + docstring），函数体留空，不写 `pass` 或 `# TODO`
- `if __name__ == "__main__":` 块中包含测试用例，格式：
  `print(f"PASS/FAIL calculate('{输入}') -> {实际输出} | expected: {预期输出}")`
- 保存后 `python xxx.py` 一次性验证所有用例
- 函数体中只用 `return`，禁止 `print`（测试块负责输出）
- 文件名使用描述性英文命名（如 `calculator.py`、`list_stats.py`），禁止 `practice1.py` 等无意义编号

## 教学规则

每次出练习题目后，必须做三件事：

1. **打开文件** — 直接将练习文件在 IDE 中打开，让学员看到函数骨架
2. **只给骨架** — 文件中只有需求注释 + 函数签名 + docstring，函数体留空，**禁止写出完整代码**；测试用例可以预先写好
3. **输出知识点清单** — 用一个表格列出本题涉及的所有知识点（类别、知识点、一句话说明），附实现顺序提示
